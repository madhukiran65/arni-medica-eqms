INTEGRATION POINTS
================================================================================

To integrate these features into RichTextEditor.jsx:

1. Import extensions:
   import { HeaderFooter } from '../../extensions/HeaderFooter'
   import { Watermark } from '../../extensions/Watermark'

2. Add to editor configuration:
   const editor = useEditor({
     extensions: [
       // ... existing extensions
       HeaderFooter,
       Watermark,
     ],
   })

3. Import and display panels:
   import HeaderFooterPanel from './HeaderFooterPanel'
   import WatermarkPanel from './WatermarkPanel'

4. Add state for panel visibility:
   const [headerFooterOpen, setHeaderFooterOpen] = useState(false)
   const [watermarkOpen, setWatermarkOpen] = useState(false)

5. Add toolbar buttons:
   <button onClick={() => setHeaderFooterOpen(true)}>Header/Footer</button>
   <button onClick={() => setWatermarkOpen(true)}>Watermark</button>

6. Render panels:
   <HeaderFooterPanel editor={editor} isOpen={headerFooterOpen} onClose={() => setHeaderFooterOpen(false)} />
   <WatermarkPanel editor={editor} isOpen={watermarkOpen} onClose={() => setWatermarkOpen(false)} />
