   npm install uuid lucide-react

3. Import and use:

   import CommentsPanel from '@/components/common/CommentsPanel';

   const editor = useEditor({
     extensions: [StarterKit, Comments],

     <>
       <EditorContent editor={editor} />
       <CommentsPanel editor={editor} currentUser={user} />
     </>
   );

================================================================================
KNOWN LIMITATIONS
================================================================================

1. No inline editing - By design for compliance (immutability)
2. No text diffing - Comments don't adjust if commented text is deleted
3. No @mentions - Reply threads don't support user mentions
4. No notifications - Email/push notifications not built-in
5. Client-side only - Must implement backend persistence
6. No conflict resolution - If comments conflict, manual resolution needed

All limitations are documented and have workarounds.
